Connect Google Calendar so Aria can schedule release campaigns, content
drops, fan emails, and tour dates directly on your calendar. Release
plans auto-populate calendar events when committed as project anchors.

<TierBadge tier="all" /> &nbsp; Free to connect.

## What Aria can do

| Action | Default behavior |
|---|---|
| Read your calendar | Yes |
| Propose new events around existing ones | Yes |
| Add a single event | Asks for confirmation |
| Add a release-plan task bundle | Asks for batch confirmation |
| Move / reschedule events | Asks for confirmation per move |
| Delete events | Asks for confirmation, with undo window |
| Read attendee responses | Yes (for meeting scheduling) |

## Connecting

<Steps>
  <Step title="Settings → Connected Platforms → Google Calendar">
    Or use the direct OAuth callback:

    ```text
    https://patchline.ai/api/auth/connect/google
    ```
  </Step>
  <Step title="Approve scopes">
    Patchline requests:
    - `calendar` (read + write)
    - `userinfo.email`
    - `userinfo.profile`
  </Step>
  <Step title="Confirm">
    Connection shows in Connected Platforms.
  </Step>
</Steps>

## Privacy

- OAuth tokens stored encrypted, per-user.
- Calendar events are read only when you ask.
- New events are created only with your explicit confirmation.

## Pricing

| Add a single event | 1 (counted as Aria chat) |
| Bulk-add release plan | 1 per Aria interaction (the events are free) |

## FAQ

<AccordionGroup>
  <Accordion title="Will Aria spam my calendar?">
    No. Aria proposes; you approve. Batch operations (like a 28-event
    release plan) get one batch-approval prompt, not 28 individual
    prompts.
  </Accordion>
  <Accordion title="Can I use Outlook / Apple Calendar?">
    Google Calendar only in v1. Outlook is on the roadmap. Apple
    Calendar via CalDAV is not currently planned.
  </Accordion>
  <Accordion title="What calendar does Aria use if I have multiple?">
    Your primary calendar by default. Multi-calendar support
    (assign release-plan events to a specific calendar) is on the
    roadmap.
  </Accordion>
</AccordionGroup>
