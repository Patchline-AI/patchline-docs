Patchline integrates with the tools you already use. Some are first-party
OAuth connections (Gmail, Calendar, Stripe). Others are read-only data
connections used for catalog import, artist intelligence, and release
planning.

<TierBadge tier="all" /> &nbsp; Integrations are free to connect. Underlying actions consume AI credits.

## How OAuth connections work

For first-party OAuth integrations (Gmail, Calendar):

<Steps>
  <Step title="Go to Settings → Connected Platforms">
    Available connections appear in the dashboard. Coming-soon
    integrations stay hidden until they are ready for self-serve use.
  </Step>
  <Step title="Click Connect on the integration">
    Browser redirects to the platform's OAuth screen.
  </Step>
  <Step title="Approve">
    Approve the requested scopes. Browser returns to Patchline.
  </Step>
  <Step title="Use it">
    Aria automatically picks up the connection. Ask Aria to do
    something — *"draft a follow-up to that curator email"* — and it
    uses the connection.
  </Step>
</Steps>

Disconnect: Settings → **Connected Platforms** → revoke. Patchline
deletes the access token within seconds.

## Built-in data connections

Some data sources power Patchline features without requiring a user
OAuth flow. For example, catalog imports use public platform metadata
where available, while artist intelligence uses licensed music and
audience data providers. These built-in connections are not something
you install; they are part of the product.

## Per-integration pricing

Most integrations are free to connect. Costs come from underlying actions:

| Action | Cost |
|---|---|
| Gmail drafted message | 1 credit (counted as Aria chat) |
| Calendar event creation | 1 credit |
| Stripe-side actions | Stripe's pricing, see [checkout](/storefront/checkout-payouts) |
| Telegram message | 1 credit |
| Spotify / Apple URL import | 0 (counts against catalog imports cap) |
